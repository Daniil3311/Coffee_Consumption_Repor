# Отчет о потреблении кофе

Этот скрипт обрабатывает CSV-файлы с данными о студентах и генерирует отчеты о расходах на кофе.

## Установка

1. Установите Python 3.8 или выше.
2. Установите зависимости: `pip install -r requirements.txt`

## Использование

Запустите скрипт с помощью следующей команды:

```
python main.py --files <file1.csv> <file2.csv> ... --report <report_type>
```

### Аргументы

- `--files`: Один или несколько путей к CSV-файлам.
- `--report`: Тип отчета для генерации. В настоящее время поддерживается: `median-coffee`

### Пример

```
python main.py --files sample.csv --report median-coffee
```

Это выведет таблицу со студентами и их медианными расходами на кофе, отсортированными в порядке убывания.

## Формат CSV

CSV-файлы должны иметь следующие столбцы:
- `student`: Имя студента
- `date`: Дата
- `coffee_spent`: Сумма, потраченная на кофе (числовое значение)
- `sleep_hours`: Часы сна
- `study_hours`: Часы учебы
- `mood`: Настроение
- `exam`: Название экзамена

## Добавление новых отчетов

Чтобы добавить новый тип отчета, создайте новый класс, наследующий от `Report`, и реализуйте метод `generate`. Затем добавьте его в словарь `reports` в `main.py`.

## Тестирование

Запустите тесты с помощью:

